# 開発ルール

## 命名規則
|title|expression|
|:--|:--|
|変数|スネークケース|
|メソッド|キャメルケース|
	
## Git
### ブランチ
|branch|explain|
|:--|:--|
|master|デプロイされているコード|
|release|リリース前の確認段階、テスト等|
|develop|開発中の統合ブランチ|
|feature/機能名|作業ブランチ、作成機能ごとに分ける|

### コミット名
|title|explain|
|:--|:--|
|[Add]|追記|
|[Fix]|修正|
|[Update]|更新|
|[Delete]|削除|
|[Move]|移動|


### Issue
制作箇所を一日作業前に書き出し
