メソッド
isBlank() : 中身がない場合trueを返す
isEmpty : 中身が空文字の場合trueを返す
isNull : 中身がnullの場合trueを返す
hashCode() : :ハッシュ値を出力する
Equals() : 参照先の比較をする、文字列やインスタンスの比較に使う
Max(a,b) : A,bの大きい方を返す
Add() : Listに引数を追加する
Get() : Listから要素を出力する
Size() : Listの要素の数を出力
Put(a,b) : Mapにキーと要素を追加するメソッド
getInstance() : Calenderクラスなどで使う、インスタンスを作成する時に使う
getTime() : calenderクラスで使う、インスタンスから日付データを取得する
printf(書式指定子, データ) : データを書式指定子に則って表示する
str.split("\\."); : 引数に正規表現を記述する
replace(“A”,”B”); : AをBに変える記述
