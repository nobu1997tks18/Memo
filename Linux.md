# Linux

## 用語
* Linuxカーネル : Linuxの祖先、無料で誰でも配布できる
* ディレクトリ : ファイルと同意
* コマンド : 命令文


## コマンド
|||
|:--|:--|
|Pwd (PrintWorkingDirectry) |現在の作業フォルダーのパスを表示|
|Cd (ChangeDirectory)|ディレクトリの移動に使う|
|Ls |カレントディレクトリの配下にあるファイルの一覧表示|
|mv 対象 移動先(新しい名前) |ファイルの移動、名前変更|
|cp 対象 ファイル名 |ファイルのコピー|
|Rm  ファイル名|削除|
|Mkdir/rmdir|ディレクトリの作成、削除|
|Vi |テキストエディタの起動|
|Cat ファイル名 |ファイルの内容を出力|
|touch ファイル名(新しいファイル名) |ファイルのタイムスタンプを変更、新規作成|
|Exit |閉じる|
|lsof -i:8080（ポート番号） |プロセスの確認|
|kill PID |プロセスの削除|
|Sassの自動コンパイル |sass --watch styles.sass:styles.css|
|tail -f  ファイル名 |ファイルの末尾を表示<br>更新があると自動でtailし直す|
